# AI Demo Icons 图片库

## 存放位置
`images/ai-demos/icons/`

## 用途说明
存放首页 AI Demo 区域各项目的图标图片

## 建议规格
- 格式：PNG（透明背景）或 SVG
- 尺寸：120x120px 或更大（保持 1:1 比例）
- 文件大小：单张不超过 100KB
- 风格：简洁、扁平化设计

## 命名规范

| 项目 | 建议文件名 |
|------|-----------|
| B2B 电商平台 | b2b-ecommerce.png |
| 电商 CRM 融合系统 | crm-fusion.png |
| BI 数据可视化平台 | bi-analytics.png |
| 企业级 CRM 系统 | enterprise-crm.png |
| 新零售电商 | new-retail.png |
| 企业品牌官网 | corporate-website.png |

## 使用方式
在首页 AI Demo 区域使用：
```html
<img src="images/ai-demos/icons/b2b-ecommerce.png" alt="B2B 电商" />
```
