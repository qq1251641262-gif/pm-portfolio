## 图标文件

### 主要图标
- `icon-main.svg` - 主图标（用于 Demo 卡片）
- `icon-16x16.png` - 浏览器标签页图标
- `icon-32x32.png` - 浏览器标签页图标（高清）

### 使用说明
1. 将设计好的图标文件放入此文件夹
2. 支持格式: SVG, PNG, ICO
3. 推荐尺寸: 16x16, 32x32, 180x180, 512x512
