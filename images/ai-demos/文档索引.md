## 🔧 数据加载器

### 文件信息
- **文件路径**: [js/ai-demos-loader.js](../js/ai-demos-loader.js)
- **类名**: AIDemosLoader
- **功能**: 从静态 JSON 加载 AI Demo 数据并提供渲染方法

### 主要方法
- `loadData()` - 加载 AI Demo 数据
- `getAllDemos()` - 获取所有 Demo
- `getDemoById(id)` - 根据 ID 获取 Demo
- `getDemoByProjectId(projectId)` - 根据项目 ID 获取 Demo
- `getActiveDemos()` - 获取激活的 Demo
- `renderDemoCard(demo)` - 渲染 Demo 卡片
- `renderAllDemoCards(containerSelector)` - 渲染所有 Demo 卡片

### 使用示例
```javascript
// 创建实例
const loader = new AIDemosLoader();

// 加载数据
await loader.loadData();

// 获取 Demo
const demo = loader.getDemoById(1);

// 渲染卡片
const cardHtml = loader.renderDemoCard(demo);

// 渲染所有卡片到容器
await loader.renderAllDemoCards('#demo-container');
```
