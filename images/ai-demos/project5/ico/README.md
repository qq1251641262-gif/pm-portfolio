# Project 5 AI Demo Icons

## 项目信息
- **项目名称**: 新零售电商解决方案
- **项目ID**: 5
- **图标用途**: AI Demo 展示区域

## 图标文件

### 主要图标
- `icon-main.svg` - 主图标（用于 Demo 卡片）
- `icon-16x16.png` - 浏览器标签页图标
- `icon-32x32.png` - 浏览器标签页图标（高清）
- `icon-180x180.png` - Apple Touch Icon

### 使用说明
1. 将设计好的图标文件放入此文件夹
2. 支持格式: SVG, PNG, ICO
3. 推荐尺寸: 16x16, 32x32, 180x180, 512x512

## 设计规范
- 主色调: #7388A3 (灰蓝色系)
- 风格: 扁平化/渐变
- 主题: 新零售、B2C、O2O
