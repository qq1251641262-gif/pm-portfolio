## 图片规格建议
- 格式：JPG 或 PNG
- 宽度：1200px 以上
- 文件大小：单张不超过 500KB
- 比例：16:9 或 4:3

## 手动添加图片步骤

1. 将图片文件放入本目录
2. 按照上表命名规范重命名文件
3. 更新 `data/projects.json` 中的图片路径（如需要）
4. 提交到 GitHub 进行部署

## 注意事项
- 图片路径在代码中引用为相对路径：`images/projects/project1/prototype/xxx.jpg`
- 确保图片文件名与 `data/projects.json` 中配置的 `image_url` 一致
