## 优化需求

### 用户要求
1. **增加卡片整体高度** - 确保内容展示完整且视觉比例协调
2. **增加图片占比** - 调整图片在卡片布局中的空间分配
3. **替换图片资源** - 使用第一张原型图替换图标
4. **增大图片尺寸** - 确保图片内容清晰可见

### 优化目标
- 提升视觉冲击力
- 增强项目展示效果
- 保持响应式特性
- 确保加载性能

## 优化方案

### 1. 卡片高度调整

#### 优化前
```html
<div class="project-card ... flex flex-col md:flex-row">
```
- 桌面端：无固定高度，由内容决定
- 移动端：无固定高度

#### 优化后
```html
<div class="project-card ... flex flex-col md:flex-row h-auto md:h-80">
```
- 桌面端：**固定高度 320px (80 * 4 = 320px)**
- 移动端：自适应高度

#### 效果对比

| 状态 | 优化前 | 优化后 |
|------|--------|--------|
| 桌面端高度 | 不固定 | 320px |
| 移动端高度 | 不固定 | 自适应 |
| 视觉统一性 | 一般 | 优秀 |
| 内容对齐 | 参差 | 整齐 |

---

### 2. 图片占比调整

#### 优化前
```html
<div class="w-full md:w-72 h-48 md:h-auto ...">
    <i class="fas fa-shopping-cart text-6xl ..."></i>
</div>
```
- 桌面端：宽度固定 288px (72 * 4)，高度自适应
- 移动端：宽度 100%，高度 192px (48 * 4)
- 内容：Font Awesome 图标

#### 优化后
```html
<div class="w-full md:w-2/5 h-64 md:h-auto relative flex-shrink-0 overflow-hidden">
    <img src="..." alt="..." class="w-full h-full object-cover ...">
</div>
```
- 桌面端：宽度占 **40% (2/5)**，高度自适应
- 移动端：宽度 100%，高度 **256px (64 * 4)**
- 内容：真实项目原型图

#### 空间分配对比

| 设备 | 优化前（图片：内容） | 优化后（图片：内容） |
|------|-------------------|-------------------|
| 桌面端 | 288px : 剩余空间 | 40% : 60% |
| 移动端 | 100% : 192px 高 | 100% : 256px 高 |
| 视觉重心 | 偏文字 | 偏图片 |

### 4. 图片显示优化

#### CSS 效果增强

**1. 悬停缩放效果**
```css
transition-transform duration-500 hover:scale-110
```
- 鼠标悬停时图片放大 10%
- 过渡时间 500ms
- 平滑流畅的动画

**2. 图片裁剪方式**
```css
object-cover
```
- 保持图片比例
- 填满容器区域
- 避免图片变形

**3. 容器溢出处理**
```css
overflow-hidden
```
- 隐藏超出部分
- 确保缩放时不溢出
- 保持圆角效果

## 响应式设计

### 桌面端（≥768px）

```css
.project-card {
    height: 320px;           /* 固定高度 */
    flex-direction: row;     /* 横向布局 */
}

.image-container {
    width: 40%;              /* 图片占 40% */
    height: auto;
}

.content {
    padding: 32px;
}
```

### 移动端（<768px）

```css
.project-card {
    height: auto;            /* 自适应高度 */
    flex-direction: column;  /* 纵向布局 */
}

.image-container {
    width: 100%;             /* 图片占满宽度 */
    height: 256px;           /* 固定高度 */
}

.content {
    padding: 24px;
}
```

## 性能优化

### 图片加载策略

**1. 懒加载准备**
```html
<img src="..." alt="..." loading="lazy">
```
- 视口外图片延迟加载
- 节省带宽
- 提升首屏速度

**2. 图片格式**
- 使用 JPG/PNG 格式
- 建议后续转换为 WebP
- 压缩优化文件大小

**3. 预加载提示**
```html
<link rel="preload" as="image" href="...">
```
- 关键图片优先加载
- 提升 LCP 指标

## 后续优化建议

### 1. 图片优化（优先级：高）

**转换为 WebP 格式**
```html
<picture>
    <source srcset="image.webp" type="image/webp">
    <img src="image.jpg" alt="...">
</picture>
```
- 减小文件体积 30-50%
- 保持相同质量

**实施懒加载**
```html
<img src="image.jpg" alt="..." loading="lazy">
```
- 首屏外图片延迟加载
- 节省带宽

### 2. 交互增强（优先级：中）

**图片点击放大**
```javascript
// Lightbox 效果
image.addEventListener('click', () => {
    // 打开灯箱查看大图
});
```

**卡片 3D 效果**
```css
.card {
    transform: perspective(1000px) rotateY(5deg);
    transition: transform 0.3s;
}
```

### 3. 性能监控（优先级：中）

**添加性能监控**
```javascript
// 监控 LCP 指标
new PerformanceObserver((entryList) => {
```
