## 🎯 快速开始

### 使用新轮播组件

1. **引入组件**
```html
<script src="js/advanced-carousel.js"></script>
```

2. **创建容器**
```html
<div id="my-carousel" class="carousel-container"></div>
```

3. **初始化**
```javascript
new AdvancedCarousel('#my-carousel', {
    images: [...],
    autoPlay: true
});
```

### 使用 AI Demo 图标

1. **准备图标文件**
   - 尺寸：120x120px
   - 格式：PNG（透明背景）
   - 位置：`images/ai-demos/icons/`

2. **在 HTML 中使用**
```html
<img src="images/ai-demos/icons/b2b-ecommerce.png" 
     alt="B2B 电商" 
     class="w-24 h-24 mx-auto">
```
