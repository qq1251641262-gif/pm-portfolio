#### 2. 修复热门页面统计

**问题**: 热门页面一直显示"加载中"

**原因**: stats.js 路由使用 SQLite 语法查询，但当前使用内存数据库

**修复**:
- 更新 stats.js 使用内存数据库 API
- 使用 memoryDB.visitor_stats 直接统计
- 实现页面访问计数和排序
- 添加今日访问量统计

**统计逻辑**:
```javascript
// 总访问量
const total = stats.length;

// 今日访问量
const today = new Date().toISOString().split('T')[0];
const todayCount = stats.filter(s => s.visit_date === today).length;

// 页面统计
const pageCounts = {};
stats.forEach(s => {
  pageCounts[s.page_path] = (pageCounts[s.page_path] || 0) + 1;
});

const pages = Object.entries(pageCounts)
  .map(([page_path, count]) => ({ page_path, count }))
  .sort((a, b) => b.count - a.count)
  .slice(0, 10);
```
